# AdventureTime


## 참고사항
 - [JayDev-GithubBrowserMVVM](https://github.com/Jeonguk-JayDev/GithubBrowserMVVM)


## 개발환경
 - M1 Mac
 - targetSDK 31
 - Android Studio Article Fox M1


## 사용기술
 - JetPack
    - compose
    - compose-navigation
    - lifeCycle
    - viewModel
 - Coroutine/Flow(비동기 작업)
 - Coil(이미지 로드)
 - Koin(Service Locator)
 - Retrofit2/OkHttp3(네트워크 통신)
 - Moshi(Json Parser)


## RESTful API 제공
 - [Sally-AdventureTimeAPI](https://github.com/sally/adventure-time-api)
